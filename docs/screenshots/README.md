# Screenshots

This directory contains screenshots and demo GIFs of the application.

## Adding Screenshots

When adding screenshots:

1. Use descriptive filenames (e.g., `homepage-desktop.png`, `login-form.png`)
2. Optimize images for web (compress, use appropriate format)
3. Include screenshots for major features
4. Consider different viewports (desktop, tablet, mobile)

## Recommended Tools

- **Screenshots**: macOS Screenshot (Cmd+Shift+4), Windows Snipping Tool, Linux Screenshot
- **Screen Recording**: QuickTime, OBS Studio, ScreenToGif
- **Image Optimization**: TinyPNG, ImageOptim, Squoosh

## Guidelines

- **File Size**: Keep images under 500KB
- **Format**: PNG for UI screenshots, GIF for animations (max 10MB)
- **Resolution**: 1920x1080 or similar for desktop, responsive sizes for mobile
- **Privacy**: Remove any sensitive information from screenshots
